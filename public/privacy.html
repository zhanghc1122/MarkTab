Privacy Policy for MarkTab

Last updated: May 5, 2026

Overview

MarkTab is a desktop Markdown editor. We respect your privacy and are committed to protecting your personal data.

Data Collection

MarkTab does not collect, store, or transmit any personal data. Specifically:

  • No personal information is collected
  • No usage analytics or telemetry
  • No advertising or tracking
  • No accounts or registration required

Local Data

All files you open and edit remain on your local device. MarkTab stores application preferences (such as font size and window position) locally on your computer. This data is never sent to any server.

Network Access

MarkTab operates entirely offline. The only network activity is an optional update check that queries the GitHub Releases API to see if a newer version is available. This request contains no personal information — only a standard HTTP request to a public API endpoint.

Third-Party Services

MarkTab does not integrate with any third-party services, SDKs, or analytics platforms.

Children's Privacy

MarkTab does not knowingly collect information from children. Since no personal data is collected at all, the application is safe for users of all ages.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated revision date.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
